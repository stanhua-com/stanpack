
## Installation
```
npm install -g stanpack
```
## Getting Started

```
stanpack <command> [options]
```

Server project packaging
```
stanpack -s
```

Weixin project packaging
```
stanpack -w
```

AntApp project packaging
```
stanpack -a
```

Project page list
```
stanpack -p
```

Statistics code line number
```
stanpack -l

```

Delete files
```
stanpack -r
```
